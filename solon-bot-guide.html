📌 이브레인봇이란?

카카오톡 오픈채팅방에서 동작하는 AI 어시스턴트입니다. 링크 자동 요약과 AI 질문 답변 두 가지 기능을 제공합니다. 이커머스·마케팅·AI 도구에 특화된 구체적 답변을 드립니다.

💬 실전 활용 예시

① 이커머스 · 쿠팡 관련 질문

입력: /이브레인봇 쿠팡 상품 상위노출 방법 알려줘
쿠팡 상위노출 핵심 방법: ① 로켓배송 등록 — 로켓 상품이 검색 우선순위 높음 ② 판매량 축적 — 초기 10~30개 리뷰가 노출에 결정적 ③ 키워드 최적화 — 상품명 앞부분에 주요 키워드 배치 (40자 이내) ④ 가격 경쟁력 — 카테고리 평균가 ±5% 이내 유지 ⑤ 클릭률(CTR) — 썸네일 첫 이미지가 흰 배경 + 상품 단독 구도 ⑥ 광고(CPC) — 초기 노출 확보 후 유기 노출로 전환 전략
입력: /이브레인봇 쿠팡 리뷰 답변을 AI로 자동화하는 방법은?
쿠팡 리뷰 AI 자동화 방법: ① ChatGPT/Claude 프롬프트 작성 — "다음 리뷰에 감사하고 공감하는 답변 3줄로 작성해줘: [리뷰 내용]" ② 구글 시트 + Apps Script 조합 — 리뷰 내용을 시트에 붙여넣기 → API 자동 호출 → 답변 생성 ③ 쿠팡 Wing API 활용 (판매자 API) — 리뷰 목록 자동 수집 가능, 답변 등록은 수동 필요 ④ 현실적 추천: Claude 프롬프트 저장해두고 복붙 반자동화 — 완전 자동보다 품질 높고 계정 정책 안전

② AI 도구 비교 질문

입력: /이브레인봇 ChatGPT와 Claude 어떤 게 더 나아?
용도별 선택 가이드: 📝 글쓰기·카피·문서 작성 → Claude (자연스러운 한국어, 긴 문맥 유지 탁월) 💻 코딩·개발·디버깅 → Claude Code 또는 ChatGPT Codex 🔍 실시간 웹 검색 필요 → ChatGPT (Browse 기능 포함) 📊 데이터 분석·Excel → ChatGPT (Code Interpreter) 🛒 이커머스 상세페이지·상품명 → Claude (톤 일관성 우수) 🖼️ 이미지 생성 → ChatGPT (DALL-E 3 내장) 결론: 텍스트 작업은 Claude, 이미지/검색 필요 시 ChatGPT

④ 마케팅 · 콘텐츠 질문

입력: /이브레인봇 스마트스토어 상세페이지 구성 방법 알려줘
스마트스토어 상세페이지 핵심 구성 순서: ① 상단 후킹 이미지 (3초 안에 구매욕 자극) — "이 제품이 필요한 사람" 공감 카피 + 강렬한 비포/애프터 ② 문제 제기 (고객 페인포인트) — "혹시 이런 경험 있으신가요?" 형식 ③ 제품 핵심 기능 3가지 (불릿 포인트) — 기능 나열 X, 해결되는 문제로 표현 ④ 신뢰 섹션 (리뷰 캡처 or 수치 데이터) — "구매자 98%가 재구매", "판매 3만개 돌파" ⑤ CTA (구매 유도) — "지금 주문하면 오늘 출고", 수량 한정 강조

❓ FAQ

Q. 봇이 응답하지 않아요

봇 운영 기기(안드로이드)가 켜져 있어야 합니다. 절전 모드 또는 앱이 종료된 경우 응답이 없을 수 있습니다.

Q. 답변이 너무 짧거나 모호해요

질문에 타겟, 상황, 원하는 형식을 추가하세요. "구체적으로", "단계별로", "예시 포함해서" 같은 지시어도 효과적입니다.

Q. URL 요약이 안 돼요

로그인이 필요한 페이지나 JavaScript로만 로딩되는 페이지는 요약이 불가합니다. 뉴스·유튜브·블로그 등 공개 페이지에서 잘 동작합니다.

Q. 한 번에 여러 질문을 할 수 있나요?

한 번에 하나씩 질문하는 게 가장 답변 품질이 좋습니다. 질문을 구체적으로 하나로 좁혀서 입력하세요.
