안드로이드 Gemini API 메신저봇R 무료

📌 솔론봇이란?

카카오톡 오픈채팅방에 참여하여 URL을 자동 요약하고 AI 질문에 응답하는 챗봇입니다.

항목내용
필요 기기안드로이드 스마트폰 (전용 기기 권장)
필요 계정카카오톡 부계정 + Google 계정
AI 엔진Google Gemini API (무료)
서버 필요 여부❌ 불필요 (폰이 서버 역할)
비용Gemini 무료 티어 활용 시 0원

🔄 동작 흐름

채팅방에 URL 또는 메시지 입력
    ↓
카카오톡 알림 발생
    ↓
메신저봇R이 알림 감지
    ↓
JavaScript 스크립트 실행
    ↓
Gemini API 직접 호출
    ↓
AI 응답 → 카카오톡 채팅방에 전송 ✅

🛠️ 사전 준비

  • 안드로이드 기기 준비 (상시 전원 연결)
  • 카카오톡 부계정 생성
  • Google AI Studio에서 Gemini API 키 발급
  • 메신저봇R 앱 설치
💡 Gemini API 키 발급: aistudio.google.com → 로그인 → Get API Key → Create API Key

📦 설치 및 설정

Step 1

메신저봇R 설치 및 권한 설정

  1. 플레이스토어에서 메신저봇R 검색 후 설치
  2. 앱 실행 → 알림 읽기 권한 허용
  3. 설정 → 접근성 → 메신저봇R 활성화
  4. 카카오톡 설정 → 알림 → 모든 알림 표시 ON
Step 2

봇 생성

  1. 메신저봇R 앱 하단 + 버튼 → 봇 이름 입력
  2. 레거시 API 사용 체크 ✅
  3. 확인 → 스크립트 편집 화면 진입
⚠️ 주의: "통합된 매개변수" 체크 해제. "레거시 API"만 체크하세요.
Step 3

스크립트 붙여넣기

기존 코드 전체 삭제 후 아래 코드 붙여넣기.

첫 번째 줄 API 키를 발급받은 키로 교체하세요.

var GEMINI_API_KEY = "여기에_API키_입력";
var BOT_TRIGGER = "봇아";
var URL_REGEX = /https?:\/\/[^\s\)\]\"\']+/g;

function response(room, msg, sender, isGroupChat, replier) {
    if (sender === "이브레인봇") return;
    var urls = msg.match(URL_REGEX);
    if (urls && urls.length > 0) {
        var url = urls[0].replace(/[.,;!?]$/, "");
        var domain = getDomain(url);
        var summary = summarizeUrl(url);
        replier.reply(domain + "(" + url + ") 에 대한 사이트 요약입니다:\n\n" + summary);
        return;
    }
    if (msg.startsWith(BOT_TRIGGER)) {
        var question = msg.replace(BOT_TRIGGER, "").trim();
        if (question.length === 0) {
            replier.reply("네! 질문을 입력해주세요 😊");
            return;
        }
        replier.reply(callGemini(question));
        return;
    }
}

function getDomain(url) {
    try {
        var match = url.match(/https?:\/\/(?:www\.)?([^\/]+)/);
        return match ? match[1].split(".")[0] : "링크";
    } catch (e) { return "링크"; }
}

function fetchUrl(url) {
    try {
        var conn = new java.net.URL(url).openConnection();
        conn.setRequestMethod("GET");
        conn.setRequestProperty("User-Agent", "Mozilla/5.0");
        conn.setConnectTimeout(8000);
        conn.setReadTimeout(8000);
        conn.setInstanceFollowRedirects(true);
        var reader = new java.io.BufferedReader(new java.io.InputStreamReader(conn.getInputStream(), "UTF-8"));
        var result = ""; var line; var count = 0;
        while ((line = reader.readLine()) !== null && count < 200) { result += line + "\n"; count++; }
        reader.close();
        result = result.replace(/<[^>]*>/g, " ").replace(/\s+/g, " ").trim();
        if (result.length > 3000) result = result.substring(0, 3000);
        return result;
    } catch (e) { return ""; }
}

function summarizeUrl(url) {
    var pageContent = fetchUrl(url);
    if (pageContent.length > 100) {
        return callGemini("다음 웹페이지 내용을 한국어로 3~4줄 요약해줘.\n형식:\n👉[페이지 제목]\n이모지 + 핵심 내용\n이모지 + 핵심 내용\n이모지 + 핵심 내용\n\n내용:\n" + pageContent);
    }
    return "URL에 접근할 수 없습니다.\n직접 확인해 보시기 바랍니다.\nURL: " + url;
}

function callGemini(prompt) {
    try {
        var apiUrl = "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=" + GEMINI_API_KEY;
        var body = JSON.stringify({contents:[{parts:[{text:prompt}]}]});
        var conn = new java.net.URL(apiUrl).openConnection();
        conn.setRequestMethod("POST");
        conn.setRequestProperty("Content-Type", "application/json; charset=UTF-8");
        conn.setDoOutput(true);
        conn.setConnectTimeout(15000);
        conn.setReadTimeout(20000);
        var writer = new java.io.OutputStreamWriter(conn.getOutputStream(), "UTF-8");
        writer.write(body); writer.flush(); writer.close();
        var code = conn.getResponseCode();
        var stream = code >= 400 ? conn.getErrorStream() : conn.getInputStream();
        var reader = new java.io.BufferedReader(new java.io.InputStreamReader(stream, "UTF-8"));
        var result = ""; var line;
        while ((line = reader.readLine()) !== null) { result += line; }
        reader.close();
        if (code >= 400) return "API 오류 " + code + ": " + result.substring(0, 200);
        return JSON.parse(result).candidates[0].content.parts[0].text;
    } catch (e) { return "오류: " + e.message; }
}

❓ FAQ

Q. 봇이 응답하지 않아요

① 메신저봇R 알림 아이콘 초록색 확인 ② 카카오톡 알림 설정 확인 ③ 봇 스위치 ON 확인

Q. API 오류가 납니다

Google AI Studio에서 API 키 재발급 후 교체. Gemini 2.5 이상 모델만 신규 키에서 동작합니다.

Q. 폰을 꺼도 되나요?

안됩니다. 봇이 실행 중인 폰은 항상 켜져 있어야 합니다. 충전기에 연결해두세요.

Q. 아이폰에서도 됩니까?

메신저봇R은 안드로이드 전용입니다. iOS 불가.
